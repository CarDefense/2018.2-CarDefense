Descrição

EVM significa Earned Value Management em inglês. Basicamente se refere a uma maneira de se fazer o cálculo do valor agregado do projeto. O cálculo do valor agregado do nosso projeto teve como base a maneira que o grupo responsável pelo projeto Dr Down fez. Tal decisão foi tomada pois eles utilizaram um padrão dentro de uma planilha automatizada.
